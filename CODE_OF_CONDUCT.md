# Contributor Covenant Code of Conduct

Como miembros de **El Séquito del Terror**, nos comprometemos a mantener un ambiente de comercio profesional.

## Compromiso
- Competencia leal entre miembros del clan.
- Respeto a las herramientas de automatización permitidas.
- Enfoque en la eficiencia financiera.
